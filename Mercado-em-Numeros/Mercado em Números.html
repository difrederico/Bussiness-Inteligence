def extrair_chave(texto): """Extrai chave de acesso (44 dígitos) do texto do QR Code, com múltiplos padrões.""" try: # Padrões comuns de URL de cupom fiscal (preservado do appscanner.py) if 'p=' in texto: return texto.split("p=")[1].split("|")[0] if 'chNFe=' in texto: return texto.split("chNFe=")[1].split("&")[0] # Buscar 44 dígitos (padrão NF-e/NFC-e) match = re.search(r'\d{44}', texto) return match.group() if match else None except Exception: return None

CAPTURA INTELIGENTE (REGEX)

O objetivo é isolar a Chave de Acesso (44 dígitos) do link completo do QR Code. O módulo utiliza **Expressões Regulares (Regex)** e lógicas de separação (`split`) para garantir a precisão da extração em diferentes formatos de cupons fiscais.

def salvar_dados(chave): """Salva chave no CSV se não existir, garantindo formato de texto.""" chave_str = "'" + str(chave).strip() + "'" # Força STRING com aspas simples if os.path.exists(ARQUIVO_CHAVES): df = pd.read_csv(ARQUIVO_CHAVES, dtype=str, encoding='utf-8-sig') # Verifica se a chave já existe (comparação como string) if 'Chave' in df.columns and chave_str in df['Chave'].astype(str).values: return False # Já existe nova_linha = pd.DataFrame({'Chave': [chave_str]}, dtype=str) df = pd.concat([df, nova_linha], ignore_index=True) else: df = pd.DataFrame({'Chave': [chave_str]}, dtype=str) # Salva o CSV com aspas em TODOS os valores (força texto) df.to_csv(ARQUIVO_CHAVES, index=False, encoding='utf-8-sig', quoting=csv.QUOTE_ALL) return True

CONTROLE E ARMAZENAMENTO (PANDAS)

O sistema verifica a existência da chave antes de salvar, evitando duplicidades. O **Pandas** é essencial para forçar o armazenamento do dado como **String** no arquivo CSV, preservando os 44 dígitos sem erro de arredondamento (truncamento numérico).
